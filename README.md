# 🎧 AUDIO PREMIUM - Cửa hàng Thiết bị Âm thanh Cao cấp

Dự án Website bán lẻ âm thanh (Loa Bluetooth, Tai nghe) hiện đại, tập trung vào trải nghiệm người dùng với giao diện Dark Mode sang trọng.

---

## 🛠️ Công nghệ sử dụng (Tech Stack)

Dự án được xây dựng trên nền tảng các công nghệ mới nhất:
- **Core:** [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/) (Tốc độ khởi động và build cực nhanh).
- **Ngôn ngữ:** [TypeScript](https://www.typescriptlang.org/) (Đảm bảo code an toàn, ít lỗi).
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Hệ thống Utility-first CSS giúp tùy chỉnh giao diện linh hoạt).
- **Icons:** [Lucide React](https://lucide.dev/) (Bộ icon thiết kế mảnh, tinh tế).

---

## 🚀 Hướng dẫn cài đặt và Chạy dự án

Để chạy dự án này trên máy cục bộ, hãy thực hiện các bước sau:

### 1. Cài đặt Node.js
Đảm bảo máy bạn đã cài Node.js (Khuyên dùng bản 18.x hoặc 20.x trở lên).

### 2. Cài đặt các thư viện (Dependencies)
Mở Terminal tại thư mục gốc của dự án và chạy:
```bash
npm install
npm run dev
